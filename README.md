# 美志愿
## 介绍
一个公益平台 APP，志愿者可以通过它来寻找感兴趣的志愿活动，交到更多志同道合的朋友。美志愿将传统繁杂的志愿活动做进手机，为传统的发布志愿活动、参与志愿项目、活动签到、工时统计、证书申请等带来便利，一切自动化完成，极大的降低了传统的人工劳动力。并在传统基础上融入社交元素，提供智能建群、圈子分享等新互联网元素。
